# ProBatch Resize – Professional Image Processor

ProBatch Resize is a high-performance, private, and powerful browser-based image resizing tool. Developed by **SoftSync Agency** for internal image processing workflows, it supports both single-image adjustments and high-speed batch processing without ever uploading your data to a server.

## ✨ Key Features

- **Batch Processing:** Resize 10, 20, or 50+ images at once and download them in a single `.zip` file.
- **Single Image Precision:** Effortlessly adjust individual photos with live previews and professional quality controls.
- **Zero-Shadow UI:** A modern, flat, and professional interface designed for focus and productivity.
- **Client-Side Processing:** All image manipulation happens locally in your browser. Your privacy is 100% guaranteed.
- **Format Support:** Export to JPEG, PNG, and WEBP with adjustable quality settings.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mahmudul-hasan-exe/probatch-resize.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
